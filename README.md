# cloud-console-test
Test project for CloudConsole team.
Project based on VueJS, Redis, ExpressJS, and NodeJS.

## Development setup

1. Open client
```bash
cd client
yarn
yarn dev
```

2. Open server
```bash
cd server
yarn
yarn dev
```

3. Open `http://localhost:3000` in browser

## Production setup

```bash
docker-compose up

# force build if necessary
docker-compise up --build
```
